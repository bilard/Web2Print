PIM Capture

Panneau flottant dans la page · Panel floating in-page
Comment ça marche :
1. Active la capture.
2. Un panneau flottant apparaît dans la page.
3. Survole un élément (cadre violet) et clique.
4. Choisis un champ (Titre, Images, …).
5. Clique ⬇ Exporter JSON dans le panneau flottant.
6. Importe le JSON dans le PIM :
  Dashboard → Templates scraping → Nouveau → Importer.
